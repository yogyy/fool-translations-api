# installation

You need [bun](https://bun.sh/) installed.

```bash
bun install
```

## Generate Migration and create Sqlite database

```bash
bun generate
```

### Run Migration

```bash
bun migrate
```

### Run Seeding (optional)

```bash
bun seed
```

### testing

make sure you're running seeding

```bash
bun test
```

### run server

```bash
bun dev
open http://localhost:4000
```
